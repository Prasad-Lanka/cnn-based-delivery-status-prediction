# CNN-Based Delivery Status Prediction

This is a beginner-level machine learning project that uses a Convolutional Neural Network (CNN) to predict food delivery status from delivery-related features.

## Project Overview

The main objective of this project is to apply a CNN model on food delivery data by converting tabular features into image-like representations. The project focuses on CNN model building, training, evaluation, visualization, and model improvement.

## Dataset

Dataset used:

`Food_Delivery_Time_Prediction.csv`

The dataset contains food delivery-related features such as:

- Delivery time
- Delivery distance
- Delivery person experience
- Restaurant rating
- Customer rating
- Order cost
- Tip amount
- Other delivery-related details

## Model Used

- Convolutional Neural Network (CNN)

## Workflow

1. Loaded and explored the dataset
2. Preprocessed the data
3. Created delivery status labels
4. Converted tabular data into image-like format
5. Built a CNN model
6. Trained and validated the model
7. Evaluated performance using classification metrics
8. Visualized model results
9. Prepared improvement and analysis reports

## Model Evaluation

The CNN model was evaluated using:

- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
- Training and validation curves

## Key Insights

- CNN models can be applied to image-like representations of tabular data.
- Training and validation curves help understand model learning behavior.
- Evaluation metrics such as precision, recall, and F1-score provide better understanding than accuracy alone.
- Model improvement techniques help compare performance before and after tuning.

## Important Note

This is a beginner-level machine learning assignment created for learning purposes. The CNN model uses image-like representations generated from tabular food delivery features. The results should be understood mainly as practice in CNN modeling, visualization, validation, and model comparison.
